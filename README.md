# Ink3D-TextureGen

Frontend website repository for Ink3D TextureGen.

## What is included

- A modern single-page static site (`index.html`)
- Custom styling and animations (`styles/main.css`)
- Interactive UI logic: tabs, tag filtering, modal, generation preview (`scripts/main.js`)
- GitHub Pages deployment workflow (`.github/workflows/pages.yml`)

## Local preview

Open `index.html` directly in a browser, or run any simple static server.

## Deploy to GitHub Pages

1. Push to `main`.
2. In GitHub, open repository settings.
3. Go to `Pages` and set `Source` to `GitHub Actions`.
4. The workflow `Deploy static site to Pages` will publish the site.
